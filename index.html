    Strength Guide

    A · B▾
    Two sessions a week, full gym, ~30 min each. You're rusty — run the first two weeks submaximal with perfect reps to re-grease the groove, then add load.

    Strength A Squat

    Warm-up
    2–3 light ramp sets on the squat
    Main
    Back squat — 3×8 · rest 90s · start ~60% of your old working weight
    Superset ×3
    Box step-ups (knee-high) 8/leg  ·  Pull-ups 5–8 (negatives if needed)
    Finisher
    Hanging leg raises 2×10–12

    Strength B Hinge

    Warm-up
    2–3 light ramp sets — drill the hinge
    Main
    Deadlift — 3×6 · rest 2 min · form before load
    Superset ×3
    Single-leg RDL (DB) 8/leg  ·  Farmer carry 40 m heavy
    Finisher
    Hanging leg raises 2×10–12 or plank 2×45s
    Add ~5 lb when every working set is clean. Bar speed stays fast — if a rep grinds or form breaks, that set's done. That's how you train rusty without getting hurt. Both lifts feed the carry: strong legs for the up, strong hinge and core for a loaded pack.
    How it shifts by phase
    Phase 1 ◂ now
    3×8–10 — rebuild & technique
    Phase 2
    Heavy: 4–5×3–5, full rest — raise the ceiling
    Phase 3
    Circuits: 3×12–15, short rest — convert to stamina
    Phase 4
    Maintenance: 2× short, no PRs — legs go to the hills
    Deload wks
    Same lifts, 2 sets, ~60%, leave 3 reps in the tank
